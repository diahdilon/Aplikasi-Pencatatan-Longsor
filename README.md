# Aplikasi-Pencatatan-Longsor

## Instalasi


Menggunakan package manager [composer](https://getcomposer.org/) untuk menginstal Laravel.

```bash
composer install
```

## Instal Plugin 

```bash
php artisan adminlte:plugins install
```

## Key Generate

```bash
php artisan key:generate
```

## Run 

```bash
php artisan serve
```
